# Tic-Tac-Toe
Designed 3 variations of Tic-Tac-Toe games where two players tries to get a row of three Xs or three Os before the other opponent does using Java and swing GUI.

Tic-Tac-Toe is a game in which two players either X or O take turns marking the spaces in a 3Ã—3 grid each trying to get a row of three Xs or three Os before their opponent does.
